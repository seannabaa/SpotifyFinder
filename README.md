## Spotify Album Finder

Utilizing Spotify's API, which can access any artist and their albums.
Programmed Using HTML, CSS, JavaScript, Node.Js, and React using a Bootstrap Framework.

---
# Functionality
- Utilizes Spotify API to find albums and more details.
- Shows artwork, release dates, and a link to the album on Spotify.

---
# Setup
Pre-reqs:
- Spotify developer account and registered application
- Client ID and Secret Client ID
- Node.js environment
